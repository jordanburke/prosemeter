# @prosemeter/readability

prosemeter readability dimensions — grade-band and per-sentence complexity

Part of [prosemeter](https://github.com/jordanburke/prosemeter) — a deterministic
scoring and feedback-loop layer for prose. See the
[main README](https://github.com/jordanburke/prosemeter#readme) for the full
picture: the agent loop, profiles, the CLI, and the MCP server.

## Install

```bash
pnpm add @prosemeter/readability
```
