# Widget Kit

Widget Kit is a small library for building dashboard widgets. It gives you a set
of ready-made charts and a simple layout grid. You can ship a dashboard in an
afternoon instead of a week.

## Install

Add the package with your usual package manager. It has no runtime dependencies,
so the install stays fast and the bundle stays small.

## Usage

Import a widget, give it some data, and drop it into the grid. Each widget takes
care of its own layout and resizing. When the window changes size, the grid
reflows the widgets to fit.

The library aims for clear defaults. Most widgets look good with no configuration
at all. When you do need to tune something, the options are named plainly and
documented in one place.

## Support

Open an issue if you hit a bug or want a new chart type. Pull requests are
welcome, and the contributing guide walks through the local setup.
