# Style Notes

Our team likes to organize the code carefully, though some files still organise things the
old way. The color palette is documented, but a few components hard-code a colour value. We
center most layouts, yet one legacy view uses a centre offset that nobody has fixed.

## Tooling

We use GitHub for reviews. Github notifications can be noisy, so configure github filters
early. The catalog of components is large; the older catalogue page is out of date and
should be archived once the new one is finished and fully reviewed by the team.
